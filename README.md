# Pirate Radio: Artifacts

**Welcome! This repository hosts the official binary releases and executables for the Pirate Radio projects.**
> The issue tracker is located [here](https://github.com/KnockoutCity-Pirate-Radio/PirateRadio-Issues).

---

### How to Install

1.  Navigate to the **[Releases Page](https://github.com/KnockoutCity-Pirate-Radio/PirateRadio-Artifacts/releases)**.
2.  Search for the latest project release (Labs or Mod Loader)
3.  Click on the "Assets" dropdown for the latest version.
4.  Download the appropriate file for your system (e.g., `.exe`, `.dmg`, `.zip`, `.tar.gz`).
